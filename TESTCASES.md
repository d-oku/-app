# テスト仕様書 (TESTCASES.md)

本ドキュメントは、音声解析アシスタント（Audio Analyst）の動作確認を行うためのテストケース一覧です。

## 1. 正常系テスト（正常に処理が完了するか）
| テスト項目 | 操作手順 | 期待される結果 |
| :--- | :--- | :--- |
| 短い音声の解析 | 1分以内の対応形式（.mp3等）を選択し、解析を実行 | 「文字起こし」「要約」「要点」の各タブに正しい結果が表示されること。 |
| タブ切り替え | 結果表示後、各タブ（文字起こし・要約・要点抽出）をクリック | 選択したタブの内容が正しく表示され、他のタブの内容は隠れること。 |
| コピー機能 | 結果表示画面で「コピー」ボタンをクリック | アクティブなタブのテキストがクリップボードにコピーされ、ボタンが「コピー完了」に変化すること。 |
| リセット機能 | 「やり直す」ボタンをクリック | 初期画面（アップロード画面）に戻り、選択されていたファイルと結果がクリアされること。 |
| 再訪時のAPIキー | ページをリロード（再読み込み）する | localStorageからキーが読み込まれ、設定モーダルが表示されずにメイン画面が使えること。 |

## 2. 異常系テスト（エラーハンドリングが機能するか）
| テスト項目 | 操作手順 | 期待される結果 |
| :--- | :--- | :--- |
| APIキー未入力 | 初回起動時、APIキーを空のまま「保存して閉じる」を押す | 「APIキーを入力してください」とアラートが出ること。 |
| 無効なAPIキー | 設定からデタラメな文字列をAPIキーとして保存し、解析を実行 | 「APIエラー: APIキーが無効です。設定を確認してください。」とエラーが表示されること。 |
| サイズ超過 | 15MB以上の音声ファイルを選択 | 「ファイルサイズが大きすぎます」と即座にエラーが表示され、解析ボタンが無効になること。 |
| 非対応ファイル | .pdf や .txt などの非対応ファイルを選択 | 「サポートされていないファイル形式です」とエラーが表示されること。 |
