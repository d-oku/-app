# Audio Analyst (音声解析アシスタント)

ブラウザ上で完結する、軽量かつセキュアな音声文字起こし・要約AIアプリです。
最新の `gemini-3.5-flash` モデルを活用し、アップロードした音声ファイルから「全文文字起こし」「3行要約」「要点抽出」を一度に自動生成します。

## 特徴
- **フロントエンドのみで完結**: HTML/CSS/Vanilla JSのみで構築。サーバー不要。
- **セキュアな設計**: APIキーはブラウザの `localStorage` に保存され、サーバーやGitHubには送信・公開されません。
- **モバイルファースト**: スマートフォンでの利用に最適化されたUI。

## セットアップと公開手順 (GitHub Pages)
1. GitHubで新しいPublicリポジトリを作成します。
2. このプロジェクトの全ファイル（`index.html`, `style.css`, `app.js`, 各 `.md` ファイル）をPushします。
3. リポジトリの `Settings` > `Pages` に移動します。
4. `Source` を `Deploy from a branch` に設定し、ブランチを `main` (または `master`) に指定して `Save` を押します。
5. 数分後、発行されたURLにアクセスしてアプリを使用できます。

## 使い方
1. アプリにアクセスすると初回のみ設定画面が開きます。
2. ご自身の [Google AI Studio](https://aistudio.google.com/app/apikey) で取得したGemini APIキーを入力して保存します。
3. 音声ファイル（.mp3, .wav, .m4aなど、15MB以内推奨）をタップまたはドラッグ＆ドロップで選択します。
4. 「解析を開始する」ボタンを押すと、AIが処理を開始します。
5. 処理が完了すると、「文字起こし」「要約」「要点」の各タブで結果を確認・コピーできます。
